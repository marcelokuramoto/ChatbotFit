------### Preparação do Ambiente ###----------
1.  NodeJS - Como instalar:
	1.1. Node Version Management para usuarios de Linux e Mac
	     Acessar github.com/nvm-sh/nvm
	     Seguir o processo da seção de "Installing and Updating"

	1.2. Nodejs.org para os usuarios de Windows
	     Realizar o Download do executavel e seguir as orientaç"oes de instalação do site

2. NPMjs- Gerenciador de pacotes do Javascript
	2.1 Será utilizado uma biblioteca do NPM para comunicação com o Telegram:
	    https://www.npmjs.com/package/node-telegram-bot-api

	2.2. Será utilizado o DialogFlow API do google
	     https://www.npmjs.com/package/dialogflow

	2.3. Será utilizado o youtube_node para facilitar a busca de video

3.DialogFlow - Ferramenta do Google para auxiliar no processo de linguagem natural
